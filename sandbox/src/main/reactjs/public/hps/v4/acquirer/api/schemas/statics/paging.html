Paging in Connect'API

A lot of the time, when you're making calls to the search API in the Connect’API, there'll be a lot of results to return. For that reason, Connect’API can paginate the results to make sure responses are easier to handle.

  • Use the pageSize (a positive value) parameter to control how many resource representations to display per page.
  • Use the pageIndex (a positive value) to display the current page's number in the pager sequence.
