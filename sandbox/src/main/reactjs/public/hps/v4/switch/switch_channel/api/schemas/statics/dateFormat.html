Supported Date Formats

The ConnectAPI supports a variety of date formats to accommodate diverse use cases and preferences. When interacting with the API, it is crucial to adhere to the specified date formats for seamless integration and accurate data handling. The supported date formats are as follows:

  • yyyy-MM-dd HH:mm:ss.SSS Z z - Example : 2023-11-14 15:30:45.123 +0300 UTC
  • yyyy-MM-dd HH:mm:ss.SSS - Example : 2023-11-14 15:30:45.123
  • yyyy-MM-dd HH:mm:ss.SS Z z - Example : 2023-11-14 15:30:45.12 +0300 UTC
  • yyyy-MM-dd HH:mm:ss.S Z z - Example : 2023-11-14 15:30:45.1 +0300 UTC
  • yyyy-MM-dd HH:mm:ss Z z - Example : 2023-11-14 15:30:45 +0300 UTC
  • yyyy-MM-dd'T'HH:mm:ss - Example : 2023-11-14T15:30:45
  • yyyy-MM-dd - Example : 2023-11-14
  • yyyyMMdd - Example : 20231114
  • yyyyMMddHHmmss - Example : 20231114153045
