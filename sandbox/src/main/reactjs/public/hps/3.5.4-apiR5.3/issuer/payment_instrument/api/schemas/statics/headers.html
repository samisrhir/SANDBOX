Headers in Connect'API

To communicate with the ConnectAPI of the PowerCard platform, you must include the proper headers in your API requests.

Name Description Request Type Required value
authorization Credentials to authenticate your request. POST,GET,DELETE,PUT,PATCH bearer JWT token
content-type The media type of the requested resource. POST,DELETE,PUT,PATCH application/json

Sample request headers

Headers: Content-Type: application/json, Authorization: bearer {_your_encoded_credentials_}
