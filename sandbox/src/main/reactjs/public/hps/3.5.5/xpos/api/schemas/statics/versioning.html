Versioning in Connect'API

  • The ConnectAPI version is indicated in each endpoint resource path.
  • For example, https://connectapi.sandbox.com/compliance/v4.0/tokens/authentication indicates the API is at version 4.0.

ConnectApi increments the version number when it releases a new version containing backwards-incompatible changes. Minor updates and bug fixes are typically released without incrementing the version number. Ensure that your application gracefully handles API revisions by allowing for additional, unspecified resource fields in the response and for variable response sizes.
